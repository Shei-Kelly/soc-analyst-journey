# Wireshark Basics

## What is Wireshark?
Network protocol analyzer used to capture and inspect traffic.

## Useful Filters

dns
http
tls
tcp
udp

## Key Fields

- Source IP
- Destination IP
- Protocol

## SOC Use Cases

- Investigating suspicious domains
- Malware analysis
- Threat hunting
- Network troubleshooting

## Key Takeaway

Wireshark allows SOC analysts to observe and investigate network communications.
