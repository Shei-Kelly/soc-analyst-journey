# Day 6 – Hour 1: Introduction to Detection Engineering

## What is Detection Engineering?

Detection engineering is the process of creating rules that identify suspicious activity and generate alerts.

## Why It Matters

Detection rules help security teams identify malicious activity hidden within large volumes of logs.

## Common Detection Categories

### Authentication
- Failed logins
- Successful logins
- Account creation

### Process Execution
- PowerShell
- Command Prompt
- Certutil

### Network Activity
- DNS
- HTTPS
- External IPs

### Persistence
- New accounts
- Scheduled tasks
- Registry changes

## Severity Levels

Low
Medium
High
Critical

## Key Insight

SOC analysts investigate alerts.
Detection engineers create them.
