# Day 4 – Hour 5: AI-Assisted SOC Investigations

## How AI Helps SOC Analysts

- Alert explanation
- Log analysis
- MITRE ATT&CK mapping
- Investigation guidance
- Incident report writing

## Key Principle

Logs provide evidence.

AI provides assistance.

Analysts must validate all findings.

## Common AI Prompts

- Explain this Wazuh alert.
- Analyze these Windows Event IDs.
- Map this activity to MITRE ATT&CK.
- Recommend investigation steps.
- Generate an incident narrative.

## Lessons Learned

AI can accelerate investigations but should never replace analyst judgment or evidence-based decision making.
