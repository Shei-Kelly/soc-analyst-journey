# Practical MITRE ATT&CK Mapping

## Tactic vs Technique

Tactic = Why the attacker acts
Technique = How the attacker acts

## Common Mappings

### 4625 Failed Login
Tactic: Credential Access

### 4624 Successful Login
Tactic: Initial Access

### 4688 PowerShell
Tactic: Execution

### 4720 User Account Created
Tactic: Persistence

### Suspicious DNS Query
Tactic: Command and Control

## Key Takeaway

MITRE ATT&CK helps SOC analysts describe attacker behavior using a common framework.
